    前言

    显然，我之前已经将树莓派搭建好了，但是呢，上一篇文章搭建有些问题。后面我一一的解释哈！我其实买个树莓派，最初的原因是我不喜欢虚拟机，真的不喜欢！现在我的树莓派搭建好了，但是出了些问题。

    问题解决

    不用写入WiFi文档

    把系统刷好过后，必须要在sd卡根目录里面写入ssh，但是不需要写入wifi的那个东东。
    直接用

    sudo raspi-config

    就直接在这个里面设置。

    更新源的问题

    更新源可能会出错。在更新源的时候，我们只用中科大的就行了。
    sources.list

    deb http://mirrors.ustc.edu.cn/raspbian/raspbian/ stretch main contrib non-free rpi
deb-src http://mirrors.ustc.edu.cn/raspbian/raspbian/ stretch main contrib non-free rpi

    下面的raspi.list可以不改，也可以改！但是你要改的话，就不要再用科大的源了。反正我出错了就是这样解决的！

    宝塔面板的安装

    为何要安装宝塔面板？
    我的答案是，我要做web服务器，还是用个面板舒服。宝塔刚好合适。

    想办法连外网（有线网络）

    我是学校的校园网，刚开始我也没办法连到外网。后来找红岩的学长，成功的解决。因为某些原因，解决方式我不会说。但是，可以说说原理！

    原理

    创翼是将我们的账号解密成正确的PPPoE账号进行拨号的，不信你可以去Windows的网络与共享中心的适配器处自己翻翻看！然后，解密算法，我不知道有没有，但是，可以想办法拦截正确的PPPoE账号，具体的我也不会。

    好，如果是路由器，那就简单了，百度一大把！

    面板安装

    就是宝塔的5.9版本，暂时只能是5.9，因为树莓派是32位的，只有5.9才支持哈！
    然后就是慢慢的等待就好了！过会儿就安装好了。

    sudo -s
sudo wget -O install.sh http://download.bt.cn/install/install-ubuntu.sh && bash install.sh
